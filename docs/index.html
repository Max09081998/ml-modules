nodejs-browser-project-template

A simple template for nodejs projects running in the browser. Include webpack, travis, jest, jsdoc and coveralls configuration.

How to use this template

  • Create a new repository using this template.

  • Open a new terminal, navigate to this project directory and run: npm install. Now you have installed the node modules.

  • To build the source file just run: npm run build
